# Project Overview

The Diabetes Risk Analysis & Healthcare Insights Dashboard is an end-to-end Healthcare Analytics and Business Intelligence project developed to analyze diabetes risk factors, patient health trends, and disease outcomes using data-driven techniques. The project focuses on transforming raw healthcare datasets into meaningful insights through data cleaning, preprocessing, exploratory data analysis, SQL analytics, and interactive dashboard development.

The solution helps identify high-risk patients by analyzing critical healthcare indicators such as glucose levels, BMI, pregnancies, insulin levels, age, and diabetes outcomes. Interactive dashboards and analytical visualizations were created to support healthcare reporting and decision-making.

# Key Features
Performed Data Cleaning and Data Preprocessing on healthcare datasets
Handled missing and invalid values using statistical imputation techniques
Conducted Exploratory Data Analysis (EDA) to identify health trends and correlations
Created correlation heatmaps, distribution plots, boxplots, and trend visualizations
Designed optimized SQL Queries for patient segmentation and diabetes analysis
Built interactive Power BI Dashboards and Tableau Reports
Generated actionable healthcare insights for risk detection and reporting
Implemented KPI reporting and dynamic analytical visualizations.
